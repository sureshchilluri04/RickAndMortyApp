Assignment Overview:
We'd like you to create a mobile app using the Rick and Morty API, which you can access here. Your task is to build an app that displays a list of 20 Rick and Morty characters. Each item on this list should show the character's name, status, and species. When a user taps on one of the list items, the app should navigate to another screen that displays the character's image and detailed information about their location, including the name, type, dimension, and number of residents.
